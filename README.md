## 🏢 About the Internship
Codveda Technology is a hub for innovation in AI/ML automation and data analysis. This internship structure requires completing **two specialized tasks per level** to master the machine learning workflow. 🎓

---

## 🟢 Level 1: Basic
*Focus: Data foundations and instance-based learning.*

### 🛠 Task 1: Data Preprocessing
* **Objective:** Prepare raw data for high-performance modeling.
* **Key Actions:** Handled missing values, performed **Label Encoding** on the Iris dataset, and implemented **Feature Scaling** (Standardization).
* **Notebook:** `Task1_Data_Preprocessing_.ipynb` 🧪

### 🔍 Task 2: K-Nearest Neighbors (KNN)
* **Objective:** Classify species based on proximity in feature space.
* **Results:** Tested multiple $k$ values ($3, 5, 7$). Achieved a peak accuracy of **96.7%**.
* **Notebook:** `Task2_Linear_Regression.ipynb` 📍 *(Note: Contains KNN implementation)*

---

## 🟡 Level 2: Intermediate
*Focus: Linear models and tree-based classification.*

### 📈 Task 1: Logistic Regression
* **Objective:** Implement a statistical model for binary classification.
* **Results:** Successfully classified species with **96.67%** accuracy. Visualized performance using **ROC Curves** and Confusion Matrices.
* **Notebook:** `Task1_ Logistic_Regression_.ipynb` 📊

### 🌳 Task 2: Decision Trees
* **Objective:** Map out decision rules for classification.
* **Results:** Achieved a perfect **100% accuracy** on the test set. Includes a detailed visualization of the tree structure.
* **Notebook:** `Task2_ Decision_Trees_.ipynb` 🌿

---

## 🔴 Level 3: Advanced
*Focus: Ensemble methods and kernel-based optimization.*

### 🌲 Task 1: Random Forest Classifier
* **Objective:** Reduce variance and improve accuracy using an ensemble of trees.
* **Results:** Achieved **93.33%** accuracy. Performed **Feature Importance** analysis to see which attributes drive predictions.
* **Notebook:** `Task1_Random_Forest_.ipynb` 🎡

### 🛡️ Task 2: Support Vector Machine (SVM)
* **Objective:** Find the optimal hyperplane for class separation.
* **Results:** Utilized the **RBF kernel** for non-linear separation, achieving **96.67%** accuracy.
* **Notebook:** `Task2_SVM_.ipynb` ⚡
